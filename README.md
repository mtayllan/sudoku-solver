# Sudoku Solver

Escrito em Ruby (2.7.1)
*[como instalar o ruby](https://linuxize.com/post/how-to-install-ruby-on-debian-10/)*

Para executar o programa basta ter o Ruby instalado e rodar o seguinte comando:

```
ruby run.rb 1 ./problem1.txt ./solucao1.txt
```

Onde `run.rb` é o nome do arquivo inicial, `1` indica o algoritmo a ser utilizado, `./problem.txt` é o arquivo do problema e `solucao1.txt` é o arquivo onde será gravado a solução.


Para mais detalhes do projeto, acesse o [REPOSITÓRIO NO GITHUB](https://github.com/mtayllan/sudoku-solver)
